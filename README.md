# zenless_auto_ahk
AHK实现绝区零（Zenless Zone Zero）自动支援和闪避
# 使用方式
第一步：安装AHK 1.1.37，详情自行搜索

第二步：“管理员身份” 运行 “zenless_auto.ahk”

第三步：启动“绝区零”，数字键4为开关
